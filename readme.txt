JazzyPay for WooCommerce - Payment plugin to easily enable faster JazzyPay checkout in your WooCommerce store.

== Description ==

### Introduction ###

JazzyPay for WooCommerce plugin integrates JazzyPay with your WooCommerce store and lets you accept payment through JazzyPay's secure payment gateway.

= About JazzyPay =

JazzyPay empower businesses of all sizes through a versatile and accessible payments platform. Learn more https://jazzypay.com/for-business

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! You can now configure the plugin.
